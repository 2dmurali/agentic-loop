## Worker Goal
Write a Python CLI calculator that evaluates basic arithmetic expressions from user input.

## Worker Instructions
- Support addition, subtraction, multiplication, and division
- Read expressions from stdin in a loop until the user types "quit"
- Handle division by zero and invalid input gracefully
- Keep it simple — no external dependencies

## Reviewer Criteria
- Correctness: Does it compute results accurately?
- Input handling: Does it handle bad input and division by zero?
- Simplicity: Is the code straightforward with no unnecessary complexity?
- Usability: Is the CLI interaction clear?
